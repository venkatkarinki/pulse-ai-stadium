The challenge

Eight friction points, every single matchday.

A 48-team tournament stresses infrastructure that was mostly built for domestic league crowds. These are the moments where a stadium visit turns stressful.

The companion

Thirteen systems, one continuous read on the stadium.

Each module runs independently but shares one live model of the venue — so a crowd spike at Gate C can trigger a shuttle reroute and a push notification at the same moment.

Live console

See what the companion sees.

The same feed, reshaped for whoever is looking at it. Switch roles to see what changes.

Wayfinding

Every gate, seat, and exit — mapped for right now.

Tap a point on the bowl. The companion answers with distance, wait, and the fastest accessible route.

Multilingual AI

Ask in any language.
Get the same clear answer.

Live speech and text translation means a volunteer's answer sounds native, no matter which of the 16 host cities — or which of the fans' home countries — it started in.
